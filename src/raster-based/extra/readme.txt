This is a staging area for scripts that were not finished during the thesis and used for data exploration purposes only.

The cloud_filter.r file is from the probaV R package tutorial by Jorn Dallinga. Its functionality is expected to be integrated into the probaV package proper in the future.

move-collection-0.sh is a shell script for moving Collection 0 files that have been reprocessed into Collection 1 to another directory; this would no longer be needed after all files are reprocessed.
